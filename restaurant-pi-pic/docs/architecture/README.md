# Kiến trúc trong `restaurant-pi-pic/docs/architecture/`

**Quy ước:** Các file **trùng tên** với `docs/architecture/` ở **root repo** (ví dụ `stack-decision.md`, `design-workflow.md`, `pi-pic-protocol.md`) là **bản mirror / làm việc trong bundle**.

- **Nguồn sự thật khi có lệch:** luôn ưu tiên chỉnh tại **`docs/architecture/`** trên root, rồi đồng bộ sang đây nếu cần.
- **Chỉ có trong thư mục này (không trùng root):** ví dụ `PRODUCT_HANDOFF.md`, `golden-demo-flow.md`, `event-mapping.md` — vẫn là tài liệu hợp lệ cho handoff/planning; khi mâu thuẫn với root, kiểm tra `docs/architecture/source-of-truth.md` trên root.
